# CricketPro – Premium Cricket Equipment E-commerce Website

A modern, responsive, full-featured cricket equipment e-commerce website built using **HTML, CSS, and JavaScript**. This project showcases a complete online shopping experience with clean UI, smooth animations, and comprehensive e-commerce functionality.

---

## 🚀 Features

### ✔ Complete E-commerce Experience
- **Multi-page website** with seamless navigation
- **Product catalog** with 97+ cricket equipment items
- **Product detail pages** with specifications and reviews
- **Shopping cart** with add/remove functionality
- **Wishlist** for saving favorite products
- **User authentication** (login/register)
- **Checkout process** with order tracking
- **Admin panel** for inventory management

### ✔ Modern & Responsive UI
- Full-width hero section with cricket stadium background
- Premium glass-blur navbar with scroll animations
- Animated buttons, hover effects, and smooth transitions
- Mobile-responsive design across all devices
- Professional payment gateway integration UI

### ✔ Product Categories
- **Cricket Bats** (English Willow, Kashmir Willow, Tennis Ball)
- **Cricket Balls** (Leather, Tennis, Practice)
- **Batting Gloves** (Professional, Youth, Practice)
- **Safety Helmets** (Professional, Youth, Wicket Keeping)
- **Batting Pads** (Professional, Youth, Wicket Keeping)

### ✔ Advanced JavaScript Functionalities
- **Real-time cart management** with localStorage persistence
- **Product filtering & sorting** (price, rating, category)
- **Search functionality** across all products
- **Toast notifications** for user actions
- **Smooth scroll navigation** and active menu highlighting
- **Scroll reveal animations** for all sections
- **EmailJS integration** for contact forms and newsletters
- **Form validation** with real-time feedback
- **Order tracking system** with status updates

### ✔ Performance & SEO Optimized
- **Lazy loading** for images
- **Comprehensive meta tags** for SEO
- **Favicon** and PWA-ready structure
- **Optimized loading** with deferred scripts
- **Responsive images** for faster loading

## ⚙️ How to Run

### Local Development
1. **Clone or download** this project folder
2. **Open `index.html`** in any modern browser (Chrome recommended)
3. **Ensure all files** are in their correct directories as shown in the structure above

### XAMPP Setup (Recommended)
1. **Place project folder** in `C:\xampp\htdocs\websites\Portfolio_Natasha_Natape\`
2. **Start XAMPP** Apache server
3. **Navigate to** `http://localhost/websites/Portfolio_Natasha_Natape/cricket-equipments/`

### EmailJS Configuration (Optional)
1. **Create EmailJS account** at [emailjs.com](https://www.emailjs.com/)
2. **Update credentials** in `styles/js/emailjs-config.js`
3. **Enable real email** functionality for contact forms
